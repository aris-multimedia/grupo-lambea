# Grupo Lambea Next.js 15 — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Bootstrap the production Next.js 15 e-commerce site for Grupo Lambea at `/Users/adria/Projects/Grupolambea/web/` with static product data, brand design system, and three pages (Home, Product Detail, Contact).

**Architecture:** Static-first App Router app — no database yet. Product data lives in `lib/productos.ts`. Shared layout uses `app/layout.tsx` with Header/Footer server components. The product detail page uses a dynamic `[slug]` route that calls `notFound()` for unknown slugs.

**Tech Stack:** Next.js 15, TypeScript, Tailwind CSS v4, shadcn/ui, lucide-react, next/font/google (Lora + Inter)
